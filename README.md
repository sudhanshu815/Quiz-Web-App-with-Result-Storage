Quiz Web App with Result Storage

🚀 A responsive and interactive Quiz Web Application built using HTML, CSS, and JavaScript. The application allows users to attempt multiple-choice quizzes, receive instant results, review answers, and save quiz history using browser Local Storage. 

📌 Project Overview

The Quiz Web App is designed to simplify the quiz-taking process by automating answer evaluation and score calculation. Users can select quiz categories, answer questions, view their results instantly, and access previous quiz attempts stored locally in the browser.  

✨ Features

👤 User Name Input

📚 Quiz Category Selection

❓ Multiple Choice Questions (MCQs)

⚡ Instant Score Calculation

📊 Percentage-Based Result Display

✅ Answer Review Section

💾 Result Storage using Local Storage

📱 Responsive User Interface

🎯 Easy-to-Use Navigation

🔄 Workflow

1. User enters their name.


2. User selects a quiz category.


3. Quiz questions are displayed one by one.


4. User selects answers.


5. System automatically calculates score.


6. Results and percentage are displayed.


7. Answer review is shown.


8. Quiz result is saved in Local Storage for future reference.
